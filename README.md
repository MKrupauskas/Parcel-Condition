# Parcel Condition
Parcel Condition was made during the Telia IoT Hackathon 2017 at Vilnius tech park
## Idea
A device you could put in a valuable package to track its
* Position 
* Temperature 
* Humidity and 
* Condition 
## App
Web app made with
* HTML5, CSS, JavaScript
* JQuery, P5.js
* And a small Python script
